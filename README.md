#### TASK 1

Реализуйте функцию `getChars(n)`, которая будет принимать целое положительное `n` и возвращать строку, состоящую из символов `x`.

_Функция должна пройти такие тесты:_
* `getChars('3'); // false`
* `getChars(-3); // false`
* `getChars(2.7); // false`
* `getChars(); // x`
* `getChars(1); // x`
* `getChars(3); // xxx`
* `getChars(7); // xxxxxxx`
